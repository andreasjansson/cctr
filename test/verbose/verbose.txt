%platform unix
===
verbose mode shows starting and completion for each test
===
cctr $CCTR_FIXTURE_DIR/tests --no-color -v 2>&1
---
starting tests/simple: test one
✓ tests/simple: test one {{ t1 }}s
starting tests/simple: test two
✓ tests/simple: test two {{ t2 }}s

✓ tests: 2/2 tests passed in {{ suite_time }}s

All 2 tests passed in {{ total_time }}s
---
where
* t1 >= 0
* t2 >= 0
* suite_time >= 0
* total_time >= 0

===
non verbose mode shows dots
===
cctr $CCTR_FIXTURE_DIR/tests --no-color 2>&1
---
..

✓ tests: 2/2 tests passed in {{ suite_time }}s

All 2 tests passed in {{ total_time }}s
---
where
* suite_time >= 0
* total_time >= 0

===
list mode shows test names
===
cctr $CCTR_FIXTURE_DIR/tests --list
---

tests
  simple: 2 test(s)
    - test one
    - test two
