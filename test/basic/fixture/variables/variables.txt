===
pattern matching with number variable
===
echo "Completed in 1.5s"
---
Completed in {{ n }}s
---
where
* n > 0
* n < 60

===
pattern matching with string variable
===
echo "Error: file not found"
---
Error: {{ msg }}
---
where
* msg contains "not found"

===
multiple variables
===
echo "Processed 42 items in 1.5 seconds"
---
Processed {{ count }} items in {{ time }} seconds
---
where
* count > 0
* time > 0
* time < 10
